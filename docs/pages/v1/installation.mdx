# Installation

Depending on your needs, _Oura_ provides different installation options:

- [Binary Release](installation/binary_release.md): to use one of our pre-compiled binary releases for the supported platforms.
- [From Source](installation/from_source.md): to compile a binary from source code using Rust's toolchain
- [Docker](installation/docker.md): to run the tool from a pre-built docker image
- [Kubernetes](installation/kubernetes.md): to deploy _Oura_ as a resource within a Kubernetes cluster
